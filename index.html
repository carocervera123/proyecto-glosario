Glosario de etiquetas HTML

01

h1

Defines a section heading of level one: the highest level.

02

a

Creates a link to a URL: a web page, a section within a page, an email address...
Also called the anchor element, where the a comes from.

03

ol

Defines an ordered list.

04

ul

Defines an unordered list.

05

p

Defines a simple paragraph of text.

06

img

Defines an image inserted in the web page.
